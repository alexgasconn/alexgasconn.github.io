01. Sobre mí

Soy Ingeniero de Telecomunicaciones con una profunda pasión por el análisis de datos y el Machine Learning. Mi curiosidad me ha llevado a especializarme en Sports Analytics, donde aplico modelos estadísticos y algoritmos para entender el rendimiento deportivo.

Con experiencia práctica en Python, SQL, y herramientas de Big Data como Spark, disfruto transformando datos crudos en visualizaciones interactivas y modelos predictivos que ofrecen respuestas claras a problemas complejos.

Actualmente estoy cursando un postgrado en Sports Analytics en la UPC junto al Barça Innovation Hub, buscando siempre nuevos retos donde pueda aplicar mis habilidades analíticas.

04. Proyectos Destacados

Strava Training Dashboard

Dashboard interactivo construido con Streamlit para visualizar y analizar actividades de Strava, con filtros por deporte y métricas clave.

  • Python
  • Streamlit
  • Pandas
  • Strava API

Player Recommender System

Herramienta que utiliza PCA y Clustering para encontrar y recomendar jugadores de fútbol con un estilo de juego similar, usando datos de FBRef.

  • Python
  • Scikit-learn
  • PCA
  • Clustering

Expected Goals (xG) Model

Modelo de regresión logística para estimar la probabilidad de gol de cada disparo en fútbol, con interpretabilidad mediante valores SHAP.

  • Python
  • Scikit-learn
  • XGBoost
  • SHAP

RaceTime Predictor

Aplicación web en Streamlit que predice tiempos de carrera para diferentes distancias usando la fórmula de Riegel.

  • Python
  • Streamlit
  • Math
